# Структура консольных команд

Почти все команды выглядят так: `git <команда> [--<параметры>] [<цели, если нужны>]`. Примеры:

`git add ZScript*.zsc`;
`git commit -a -m "Commit"`;
`git checkout master`;
`git stash push`;
`git rm --cached *.h`;
